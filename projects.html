Using SVD for Face Detection

I worked on a fun project that involved implementing image-processing tasks using linear algebra concepts on The Yale Face Database. The project consisted of four parts. In the first part, I extracted eigenfaces. The second part involved compression using Singular Value Decomposition. The third part involved compression using the Discrete Cosine Transform. Finally, in the fourth part, I implemented face detection using SVD.
